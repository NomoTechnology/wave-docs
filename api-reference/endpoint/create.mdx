---
title: "Create Customer"
api: "POST /customers"
description: "Este endpoint cria um novo cliente"
---

### Body

<ParamField body="phone" type="string" required>
  Número de telefone incluindo o country code. 55, no caso do Brasil.
</ParamField>

<ParamField body="name" type="string">
  Nome do cliente.
</ParamField>

<ParamField body="email" type="string">
  E-mail do cliente.
</ParamField>

<ParamField body="taxId" type="string">
  Identificador fiscal. CPF ou CNPJ, no caso do Brasil.
</ParamField>

<ParamField body="metadata" type="object">
  Parâmetro de uso livre pelo integrador.
</ParamField>

<ParamField body="subscriptionMetadata" type="object">
  Parâmetro de uso livre pelo integrador.
</ParamField>

### Response

201 CREATED
