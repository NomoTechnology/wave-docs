---
title: "Delete Customer"
api: "DELETE /customers/{phoneNumber}"
description: "Remove um cliente existente. Após ser removido, um cliente não pode ser reativado."
---

<ParamField path="phoneNumber" type="string">
  Telefone do cliente. Apenas números e incluindo o country code.
</ParamField>

### Response

200 OK

<RequestExample>

```bash Request
curl --location --request DELETE '<BASE_URL>/customers/:phone_number' \
--header 'Content-Type: application/json' \
--header 'Authorization: Token <token>'
```

</RequestExample>

<ResponseExample>

```json Response 200
200 OK
```

</ResponseExample>
