---
title: "Block Subscription"
api: "POST /customers/{phoneNumber}/subscriptions/block"
description: "Desativa a renovação da subscription de um cliente. A subscription permanecerá ativa até o final do ciclo existente."
---

<ParamField path="phoneNumber" type="string">
  Telefone do cliente. Apenas números e incluindo o country code.
</ParamField>

### Response

200 OK

<RequestExample>

```bash Request
curl --location --request POST '<BASE_URL>/customers/:phone_number/subscriptions/block' \
--header 'Content-Type: application/json' \
--header 'Authorization: Token <token>'
```

</RequestExample>

<ResponseExample>

```json Response 200
200 OK
```

</ResponseExample>
