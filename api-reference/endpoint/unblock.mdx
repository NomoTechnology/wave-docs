---
title: "Unblock Subscription"
api: "POST /customers/{phoneNumber}/subscriptions/unblock"
description: "Reativa a renovação da subscription de um cliente. Se já existe um ciclo corrente, um novo ciclo não será criado."
---

<ParamField path="phoneNumber" type="string">
  Telefone do cliente. Apenas números e incluindo o country code.
</ParamField>

### Response

200 OK

<RequestExample>

```bash Request
curl --location --request POST '<BASE_URL>/customers/:phone_number/subscriptions/unblock' \
--header 'Content-Type: application/json' \
--header 'Authorization: Token <token>'
```

</RequestExample>

<ResponseExample>

```json Response 200
200 OK
```

</ResponseExample>
