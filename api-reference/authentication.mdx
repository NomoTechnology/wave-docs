---
title: "Autenticação"
description: "Definição do método de autenticação utilizado pela API"
---

Autentique-se na API da Jaw Games solicitando uma Chave de API ao nosso time de integração.


A Jaw Games utiliza autenticação por chaves de API porque são uma forma segura de se comunicar com a API da Jaw Games. Se sua chave de API for comprometida, é fácil desativá-la e criar uma nova para que você possa atualizar suas variáveis de ambiente com a nova chave. As permissões da chave de API possuem um acesso mais amplo que as credenciais normais de usuário.

Para usar uma chave de API, passe um cabeçalho `Authorization` com um valor de `Bearer <Sua-Chave-de-API-Aqui>`, onde você substitui `<Sua-Chave-de-API-Aqui>` pela chave de API que você recebeu do nosso time de integração.

<CodeGroup>
```
GET https://api.jaw.games/customers/5585999999999 HTTP/1.1
Authorization: Bearer <Sua-Chave-de-API-Aqui>
```
</CodeGroup>
##
<CodeGroup>
```shell
curl -X "GET" "https://api.jaw.games/customers/5585999999999" -H "Authorization: Bearer <Sua-Chave-de-API-Aqui>" -H "Content-Type: application/json"
```
</CodeGroup>
##
<Warning>Mantenha sua chave de api em sigilo, não compartilhe com ninguém, são dados sensíveis aos quais apenas você deve ter acesso.</Warning>

<Note>Para autenticar na nossa API, você vai precisar de um chave de api válida. Entre em contato com o nosso time de integração para solicitar a sua credencial.</Note>
